# PDF Analyzer AI 🚀
An AI-powered web application that analyzes PDFs, generates summaries, extracts keywords, and creates exam-focused questions.

## Features
- AI-powered note summarization
- PDF and text support
- Clean and responsive UI
- Fast summarization

## Tech Stack
- Next.js
- TypeScript
- AI API
- Vercel
- 
## Development Journey
Initially built using Python, later migrated to Next.js to improve UI, deployment, and overall user experience.
