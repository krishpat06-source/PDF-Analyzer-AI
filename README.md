# AI Notes Summarizer 🚀

An AI-powered web application that summarizes notes and PDFs into concise, easy-to-understand summaries.

## Features
- AI-powered note summarization
- PDF and text support
- Clean and responsive UI
- Fast summarization

## Tech Stack
- Next.js
- TypeScript
- AI API
- Vercel
-

Initially built using Python, later migrated to Next.js to improve UI, deployment, and overall user experience.
